h1. Filter Parameter Logging

This extension adds parameter filters to your "radiant":http://github.com/radiant/radiant (production) log.

Tested with 0.8.x and 0.9.x

h2. Installation

Starting from your radiant application root directory:

git clone git://github.com/jfqd/radiant-filter_parameter_logging-extension.git vendor/extensions/filter_parameter_logging

No rake tasks needed.
